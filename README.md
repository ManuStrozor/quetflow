# QuetFlow

**QuetFlow** est une application web basée sur Google Apps Script permettant de centraliser, transformer et visualiser les données de quêtes paroissiales à partir de fichiers d'export (CSV, Excel).

## 🎯 Objectif

Faciliter le travail des responsables comptables en automatisant le traitement des données issues de terminaux de collecte (quêtes), en l'absence d’API directe.

## ⚙️ Fonctionnalités

- 📥 Import de fichiers CSV / XLSX issus de systèmes externes
- 🔄 Transformation et normalisation des données
- 📊 Génération de tableaux croisés dynamiques
- 📈 Visualisation des indicateurs clés via une interface web
- 🧮 Calculs automatisés pour le suivi financier des quêtes

## 🏗️ Architecture

- **Google Apps Script**
- **Google Sheets** (stockage + calculs)
- **Web App HTML/JS** (dashboard)

## 👥 Utilisateurs cibles

- Responsables comptables de paroisses
- Structures religieuses gérant des flux de dons

## 🚀 Usage

1. Importer un fichier d’export (CSV/XLSX)
2. Laisser QuetFlow traiter et structurer les données
3. Consulter les indicateurs et tableaux de bord via la web app

## 💡 Contexte

Les données sont issues de terminaux de collecte utilisés lors des célébrations religieuses, récupérées via export manuel depuis un service externe ne proposant pas d’API.
